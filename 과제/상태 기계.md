### 1. 캐릭터 전직
* **가질 수 있는 상태**
    1. 현재 직업 2. 직업 선택 3. 전직 완료
* **상태가 변경되는 조건**
    1.  현재 직업 -> 직업 선택 -> 전직 완료
        * 현재 직업과 다른 직업 선택 시 전직 완료

### 2. 엘리베이터
* **가질 수 있는 상태**
    1. 정지 2. 운행 3. 고장 4. 수리
        * 1. 정지: 기본 상태, 수리 중, 고장 시
* **상태가 변경되는 조건**
    1.  정지 -> 운행
        * 누군가 층 수 클릭
    2.  운행 -> 대기
        * 층 도착
    3.  운행 -> 고장
        * 엘리베이터 내부 부품 고장 등
    4.  고장 -> 정지
        * 엘리베이터 고장 날 시 엘리베이터 정지
    5.  고장, 정지 -> 수리
        * 엘리베이터가 고장나서 정지하면 수리

### 3. TV
* **가질 수 있는 상태**
    1. 미작동 2. 작동 3. 고장 4. 수리
* **상태가 변경되는 조건**
    1.  미작동 -> 작동
        * 누군가 작동 리모컨 누를 시
    2.  작동 -> 미작동
        * 누군가 미작동 리모컨 누를 시
    3.  작동 -> 고장
        * TV가 외부의 충격을 받을 경우 등
    4.  미작동, 고장 -> 수리
        * TV가 고장나서 정지하면 수리
